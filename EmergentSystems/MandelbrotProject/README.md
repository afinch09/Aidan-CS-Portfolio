Fractal Project:

The Mandelbrot and Julia sets and sets of complex numbers defined by the convergance of each value based on particular rules
Both sets have recursive rules that go to infinity and if a series converges it is a member of the set
Graphically these results are dictated by color, if a point is black the point converges
If a point is divergent it is colored based on the normalized value of how many loops it took the point to diverge

The rule for the mandelbrot set is the following:
z_(n+1)=(z_n)^2+c and z_0 = 0 and c is a point in the complex plane

The rule for the julia set:
z_(n+1)=(z_n)^2+c where z_0 is a number in the complex plane and c is a fixed number in the complex plane
The c value chosen has drastically different effects on the resulting image and can create vivid imagery if chosen correctly
Particular c values I found interesting have been stored and can be selected at run time along with the option to choose your own

Code is supplied in a seperate folder
To run you must have a jdk installed previously
Create a java project
Place all code into the source file
Run the fractal panel file
