Gate neural network:

I used numpy to create a simple neural network that learns simple logic gates
I used backpropagation for computing gradients, binary cross-entropy loss for training, sigmoid activation which introduces non-linearity, and gradient descent to minimize loss over time

Also added custom hidden layer size, learning rates, training epochs, and visualization with matplotlib
This can be used as an exploration for the basics behind machine learning and how different choices effect the learning process and behavior of a model

How to run:
pip install numpy matplotlib
python gate.py
